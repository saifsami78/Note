## 👁️ Preview in VS Code

| Action | Shortcut |
|--------|----------|
| Open preview (side by side) | `Ctrl + K` then `V` |
| Open preview in current tab | `Ctrl + Shift + V` |

Or right-click the `.md` file tab → **Open Preview**.

---

## Headings

```md
# H1 — Page Title
## H2 — Section
### H3 — Subsection
#### H4
```

> Always put a space after `#`

---

## Text Formatting

```md
**bold**
*italic*
***bold and italic***
~~strikethrough~~
`inline code`
```

---

## Lists

```md
- Item
- Item
  - Nested item (2 spaces)

1. First
2. Second
3. Third
```

---

## Links & Images

```md
[Link Text](https://example.com)

![Alt text](image.png)
```

---

## Blockquotes

```md
> This is a blockquote.
```

---

## Code

````md
`inline code`

```python
print("Hello!")
```
````

Common language tags: `python` `javascript` `html` `css` `bash` `json`

---

## Tables

```md
| Name  | Age |
|-------|-----|
| Alice | 25  |
| Bob   | 30  |
```

---

## Task Lists

```md
- [x] Done
- [ ] Not done
```

---

## Horizontal Rule

```md
---
```

---

## Escaping Special Characters

Add `\` before a character to use it literally:

```md
\*not italic\*
\# not a heading
```

---

## 💡 Two Things Worth Remembering

- **Blank line** between paragraphs = new paragraph
- **Two spaces** at end of a line = line break
